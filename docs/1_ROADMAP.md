# Phase 1: Product Roadmap

## Overview
**Unheard India** is an ethnographic research platform designed to document and preserve the cultural heritage of India's marginalized communities. This roadmap outlines the development journey, current status, and future goals of the platform.

## üèÅ Completed Milestones (Phase 1-6)

### Q4 2025 - Q1 2026: Foundation & MVP

#### Core Platform
- [x] **Project Setup**: Initialized Next.js frontend and Express backend repositories.
- [x] **Database Design**: Implemented MongoDB schemas for Users, Communities, and Research data.
- [x] **Authentication**: Secure JWT-based registration and login system.
- [x] **Role-Based Access Control**: Differentiated access for public users, students/researchers, and admins.

#### Content & Communities
- [x] **Community Documentation**: Detailed profiles for Gadia Lohar and Bhoksa communities.
- [x] **Multimedia Integration**: Image galleries and documentary video embedding.
- [x] **Search & Filtering**: Ability to filter communities by region and search by name.

#### Research & Education
- [x] **Research Data Management**: System for submitting and viewing ethnographic research papers.
- [x] **Student Submissions**: Form interface for students to submit fieldwork findings.
- [x] **Ethics Guidelines**: Dedicated section for research ethics.

#### Admin & Management
- [x] **Admin Dashboard**: Interface for managing content, users, and approvals.
- [x] **File Management**: Multer-based upload system for images and documents.
